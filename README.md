## 🔧 Trouble Shooting

### 1. 로그인 사용자 정보 가져오기 문제
- **문제:** 게시글 작성 시 로그인 사용자 정보를 게시글 작성자 정보로 저장해야 했음
- **원인:** Spring Security에서 사용자 정보를 여러 방식으로 가져올 수 있어 혼란 발생
- **해결:** `SecurityContextHolder`에서 `Authentication` 객체를 가져와 `CustomUserDetails`로 캐스팅
  
### 2. 게시글-댓글 관계 설계 문제
- **문제:** 댓글이 어느 게시글에 속하는지 구분이 필요
- **원인:** 댓글 테이블만으로는 특정 게시글 댓글 조회가 어려움
- **해결:** 댓글 테이블에 `board_id` 추가하여 1:N 관계 설정
- **결과:** 특정 게시글 댓글 조회 및 관리가 용이해짐

### 3. 이미지 업로드 깨짐 문제 (미해결)
- **문제:** 게시글 작성 시 업로드한 이미지가 화면에서 깨져 표시됨
- **원인:** 서버 저장 경로와 브라우저 접근 경로 불일치
- **시도한 방법:** 업로드 경로를 `application.properties`에서 관리, 게시글 데이터에 `savePath` 저장
- **결과:** 파일 서버 저장 및 DB 저장은 정상, 브라우저에서 이미지 깨짐 문제는 미해결
- **배운 점:** 파일 업로드 시 서버 저장 경로와 웹 접근 경로를 함께 고려해야 함. 추후 `ResourceHandler`로 개선 예정
